# Munazir Ansari - Portfolio Website

A futuristic, highly interactive 3D portfolio website showcasing my journey as an aspiring AI/ML Engineer, Frontend Developer, and UI/UX Designer.

## 🚀 Features

- **Interactive 3D Hero Section** - Built with Three.js and React Three Fiber
- **Smooth Animations** - Powered by Framer Motion
- **Glassmorphism Design** - Modern UI with glass effects and neon accents
- **Fully Responsive** - Optimized for all devices
- **Performance Optimized** - Fast loading and smooth interactions
- **Custom Cursor** - Interactive cursor with trail effect
- **Scroll Progress Indicator** - Visual feedback for page navigation

## 🛠️ Tech Stack

- **React.js** - Frontend framework
- **Three.js / React Three Fiber** - 3D graphics and animations
- **Framer Motion** - Animation library
- **Tailwind CSS** - Utility-first CSS framework
- **Vite** - Build tool and dev server
- **Lucide React** - Icon library

## 📦 Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd munazir-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

5. Preview production build:
```bash
npm run preview
```

## 🎨 Customization

### Colors
Edit `tailwind.config.js` to customize the color scheme:
```javascript
colors: {
  primary: '#00f0ff',    // Cyan
  secondary: '#ff00ff',  // Magenta
  accent: '#7000ff',     // Purple
  dark: '#0a0a0f',       // Dark background
  darker: '#050508',     // Darker background
}
```

### Content
Update the content in each component file:
- `src/components/Hero.jsx` - Hero section
- `src/components/About.jsx` - About section
- `src/components/Skills.jsx` - Skills section
- `src/components/Projects.jsx` - Projects section
- `src/components/Achievements.jsx` - Achievements section
- `src/components/Contact.jsx` - Contact section
